# Assets

Esta pasta centraliza os recursos visuais e sonoros do jogo.

## Organização

- `imagens/`: sprites da cobra, maçã e outros visuais.
- `sons/`: efeitos sonoros.
- `fontes/`: fonte tipográfica do jogo.
